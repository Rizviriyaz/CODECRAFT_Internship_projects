
# Full Guide: Temperature Converter (Frontend + Backend)

This repository contains a **Node.js/Express** backend API and a **vanilla HTML/CSS/JS** frontend.

## Quick Start

### 1) Backend
```bash
cd backend
npm install
npm run dev   # or npm start
# API at http://localhost:3000
```

### 2) Frontend
Open `frontend/index.html` in your browser.
(Or serve the folder with any static server.)

### 3) Test
Try converting 25°C. You should see Fahrenheit and Kelvin values.
