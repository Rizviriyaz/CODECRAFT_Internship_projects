## ⚙️ Setup & Installation

Each project contains its own `README.md` with specific setup instructions. Generally:

- For **Python** projects: `pip install -r requirements.txt` (if applicable) and run the `.py` file.
- For **Node.js** projects: Navigate to the `backend/` directory, run `npm install`, and start with `npm run dev` or `node server.js`.
